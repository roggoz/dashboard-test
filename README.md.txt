# coalition test
this is a react based project
I have used react+vite to build this project

## installation
to set up the project, follow this steps:

# Navigate to the project directory
cd coalition-test

#install vite+react using npm
npm install vite@latest

# Install dependencies using npm
npm install

# Start the development server
npm run dev
